# Request Support with AWS Support

## Overview

In this lab, you will be introduced to the AWS support service, the support plans of AWS Support, and learn how to access and create a support request on AWS Support.

## AWS Support

AWS Support is available in various support plans that provide access to tools and varying levels of expertise to support the operation and success of solutions built on the AWS platform. All support plans grant you 24/7 access to customer care, AWS documentation, technical documentation, and support forums.

Depending on your technical support requirements and the resources needed to develop your AWS environment, you should choose a support plan that aligns with your current AWS solution needs.
